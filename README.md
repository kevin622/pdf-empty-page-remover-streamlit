# PDF 빈 페이지 제거기

## 프로젝트 소개

이 프로젝트는 **PDF에서 빈 페이지를 자동으로 제거**해주는 간단한 웹 도구입니다.
PDF를 업로드하고 버튼을 누르면, 빈 페이지를 제거한 결과물을 즉시 다운로드할 수 있습니다.

Streamlit 기반이라 별도 UI 개발 없이도 누구나 브라우저에서 쉽게 사용할 수 있고, Python 환경만 준비되어 있으면 실행이 간단합니다.

## 주요 기능

- PDF 업로드 → 빈 페이지 제거 → 결과 PDF 다운로드 (한 화면에서 끝)
- 빈 페이지 판단 기준을 **텍스트/이미지/주석**으로 확인
- 처리 결과 요약(총/삭제/유지 페이지 수) 표시

## 요구 사항

- Python (프로젝트 설정은 `pyproject.toml`의 `requires-python` 참고)
- `uv`

## 설치 및 실행

1. 의존성 설치

`uv sync`

2. 앱 실행

`uv run streamlit run main.py`

3. 사용

브라우저가 열리면 PDF 파일을 업로드하고 **빈 페이지 제거 실행**을 누른 뒤 다운로드 버튼으로 결과 파일을 받습니다.

## 동작 방식(빈 페이지 판단 기준)

각 페이지에 대해 아래 요소를 확인합니다.

- 텍스트: `extract_text()` 결과가 비어있지 않으면 유지
- 주석: 하이라이트/메모 등의 주석(Annots)이 있으면 유지
- 이미지: 페이지 리소스의 XObject 중 Subtype이 Image가 있으면 유지

위 요소가 **모두 없는 경우**에만 “완전히 비어있는 페이지”로 판단하여 삭제합니다.

## 제한 사항 / 참고

- 비밀번호로 보호된 PDF 또는 손상된 PDF는 처리 중 오류가 발생할 수 있습니다.
- 스캔본처럼 이미지로만 이루어진 페이지는 이미지가 감지되므로 일반적으로 삭제되지 않습니다.
- PDF 내부 구조가 매우 특이한 경우(일부 리소스 파싱 실패 등)에는 감지가 완벽하지 않을 수 있습니다.

## 개발 메모

- 핵심 처리는 메모리(BytesIO)에서 수행해 디스크 I/O 없이 동작합니다.
- Streamlit UI는 단일 파일(`main.py`)로 구성되어 있어 배포/실행이 단순합니다.
